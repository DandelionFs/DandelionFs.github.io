说实话中文字体界实在是太乱了, 在使用过Google Fonts之后一直想找替代的网站, 简中搜索结果总是着指向诸如xxx下载站, pc下载站, 第一字体网之类的质量十分低下(缺乏字体介绍, 使用许可, 作者详情, 官方网站等)的网站, 然而他们的 SEO 偏偏总是最高的, 这种资源的不平衡让我有点受不了, 这和人们的使用习惯, 闭源的运营方式, 严格的审查制度有着十分密切的关系.

我们总是不把版权问题摆在前面, 总是先入为主的要使用某一件产品而不问有什么代价, 总是乐于做伸手党, 久而久之真的沦为了理所当然的事情, 因为每一片雪花都很小, 所以雪崩不怪任何人, 甚至伸手党们拉帮结伙开始反攻作者, 真的让人啼笑皆非, 下面是微软雅黑的一些使用版权问题, 望周知.

微软公司斥巨资委托方正字体公司专门为Vista系统设计制作了微软雅黑(方正兰亭黑), 微软公司只拥有微软雅黑的使用权, 而版权在方正手中. 我们仅可以在Windows系统中使用微软雅黑！脱离Windows平台的一切商业行为, 都属于侵权行为, 同时在Windows系统下, 商业行为也不被允许, 包括网页设计中, 主动调用微软雅黑字体, 更不可以以图片的形式在网页中用到微软雅黑！1

BASIC THINGS

  • 磅 & 字号: 1磅约等于 0.35毫米, 即一个10磅的文字, 其高度约等于3.5毫米

  • 衬线指的是字母结构笔画之外的装饰性笔画. 有衬线的字体叫衬线体（serif）；没有衬线的字体, 则叫做无衬线体（sans-serif）2, 个人感觉衬线字体阅读感更强, 无衬线更美化.

Mono/等宽字体

挑选字体方面, 需留意对字体编码 Unicode 要优先考虑简中, 否则会出现字体变形的难受字样.

版权方面, 微软雅黑是从Window上面提取的版本, 所以存在版权问题. 只可以个体使用, 不可用于商业环境; 更纱黑体由 兆邦中国(Zhaobang China) 在微软商店免费下载使用.

  1. Microsoft Yahei Mono | 微软雅黑官方等宽字体 .
  2. Sarasa Gothic / 更纱黑体 / 更紗黑體.
  3. FiraCode | 符号连体.
  4. Adobe Fonts-Source Code Pro | 全英文环境较好.

BUGS

中竖门 & 门

微软雅黑默认状态下部分字符显示为日文字符, 如门就会显示 '中间一竖的门'. 纠正方法换字体 --文泉译米 即可

Profromence differences are related with CJK Unified Ideographs, and relating to CJK Font. And Some Font display the Japan Font as default pattern, such this Mircosoft YaHei in ubuntu when the sys encoding is US_Utf-8, but why the window is not such situation?3. Some more details FAQ with it. By th way, Japan encodes CIJ and Chinese encodes GB.
